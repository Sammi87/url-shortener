# Url-shortener
An application to shorten urls

## Installing

### Run in frontend:

`npm install` 

### Run in backend: 

`npm install` 


### Start MongoDB:

#### Install
`brew tap mongodb/brew`

`brew update`

`brew install mongodb-community@7.0`

#### Run
`brew services start mongodb-community@7.0`


## Running Url-shortener

### Run in frontend

`npm run start`

### Run in backend

`npm run start`

After this, visit http://localhost:9000
